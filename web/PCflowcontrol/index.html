About this Project


FlowControl Automatic Plant Watering System เป็นโปรเจกต์ IoT ที่ใช้ Arduino Uno R4 WiFi ในการตรวจสอบระดับความชื้นในดินและรดน้ำต้นไม้อัตโนมัติเมื่อจำเป็น ผู้ใช้สามารถตรวจสอบและควบคุมระบบจากระยะไกลผ่านเว็บอินเทอร์เฟซ ช่วยลดการใช้น้ำและเพิ่มความสะดวกสบายในการดูแลต้นไม้

การรดน้ำอัตโนมัติ: ระบบใช้เซ็นเซอร์ตรวจวัดความชื้นในดินเพื่อตรวจสอบความต้องการน้ำของต้นไม้ และทำการเปิดปั๊มน้ำเมื่อระดับความชื้นลดลงต่ำกว่าค่าที่กำหนด การตรวจสอบและควบคุมจากระยะไกล: ด้วยการใช้ Arduino ที่เชื่อมต่อ WiFi ผู้ใช้สามารถเข้าถึงข้อมูลของต้นไม้และควบคุมกระบวนการรดน้ำได้จากทุกที่ เพิ่มความสะดวกสบายและความอุ่นใจ
การใช้น้ำอย่างมีประสิทธิภาพ: โดยการรดน้ำเฉพาะเมื่อจำเป็น ระบบนี้ช่วยลดการใช้น้ำอย่างสูญเปล่า ทำให้เป็นมิตรต่อสิ่งแวดล้อมและประหยัดค่าใช้จ่าย
